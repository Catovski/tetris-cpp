# Console Tetris in C++

A classic Tetris game implemented in C++ for the console/terminal.

![Game Preview](screenshots/gameplay.gif)

## Features
- 7 classic Tetris pieces
- Score system with levels
- Next piece preview
- Simple and clean interface
- Cross-platform (Windows/Linux/Mac)

## Installation & Compilation

### Windows
```bash
g++ main.cpp -o tetris.exe
./tetris.exe
